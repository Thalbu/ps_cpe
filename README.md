# ps_cpe
No projeto simulado da Consultoria e Projetos Elétricos (CPE) da UFMG foi desenvolvido por mim e outros 3 integrantes um site básico para exposição de cardápio de um chefe de cozinha que trabalho com cupcake's. O site deveria conter uma pagina apenas e todos feito em HTML 5, CSS 3 e Java Script, além de uma parte de contatos no final que realizasse o envio de e-mails automáticos.
